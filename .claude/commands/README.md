# Commands

Slash commands for common development tasks.

## Available Commands

| Command | Description |
|---------|-------------|
| `/audit` | Run comprehensive code audit (TS, lint, security, coverage) |
| `/deps` | Check and update project dependencies |
| `/dev-check` | Run lint, type-check, and build verification |
| `/docs` | Create documentation or guidance files |
| `/generate-bookmark` | Generate bookmark-related code (component, hook, test, type) |
| `/refactor` | Refactor code while maintaining functionality |
| `/test` | Run tests and report results |

## Usage

Type `/` in the chat to see available commands, or invoke directly:

```
/audit
/deps
/dev-check
/docs
/generate-bookmark component Header
/generate-bookmark hook useSearch
/generate-bookmark test BookmarkCard
/generate-bookmark type FilterOptions
/refactor
/test
```
