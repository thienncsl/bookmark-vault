# Agents

AI agents specialized for complex, multi-step tasks.

## Available Agents

### Bookmark-Specific

| Agent | Description |
|-------|-------------|
| `architecture-planner` | Design implementation plans for new features |
| `bookmark-coordinator` | Orchestrate multiple analysis tasks simultaneously |
| `bookmark-data-analyst` | Analyze state management, localStorage, and data flow |
| `bookmark-reviewer` | Review code for React/TS/localStorage best practices |
| `bookmark-ui-analyst` | Analyze component structure, styling, accessibility |

### General Purpose

| Agent | Description |
|-------|-------------|
| `react-reviewer` | Specialized React/Next.js code review |
| `security-reviewer` | Security vulnerability assessment |
| `test-builder` | Create and update tests for components/utilities |

## Usage

Agents are invoked via the Task tool for complex tasks that require:
- Multiple analysis perspectives
- Iterative problem-solving
- Deep exploration of the codebase
- Planning and implementation
