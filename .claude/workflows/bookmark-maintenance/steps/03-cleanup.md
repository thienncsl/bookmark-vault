## Suggest Cleanup Actions

Based on validation results, suggest fixes.

## Cleanup Categories

### Auto-fixable
- Add missing protocol to URLs (http:// or https://)
- Trim whitespace from titles
- Remove duplicate bookmarks (keep oldest)

### Manual Review Required
- Empty descriptions (suggest based on URL content)
- Missing tags (suggest based on domain patterns)
- Suspicious URLs (check if accessible)

## Output Format

Prioritized list of suggested actions with effort estimate.
