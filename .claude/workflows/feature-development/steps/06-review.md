# Generate Implementation Summary

Create a summary of the completed feature implementation.

## Input

From previous steps:
- Feature description: $FEATURE_DESCRIPTION
- Implementation steps: $IMPLEMENTATION_STEPS
- Files affected: $FILES_AFFECTED
- Checks pass: $CHECKS_PASS

## Tasks

1. **Summarize changes**
   - What was implemented?
   - What files were changed?
   - What tests were added?

2. **Document key decisions**
   - Architecture choices
   - Pattern decisions
   - Trade-offs made

3. **Note next steps**
   - Any follow-up tasks
   - Future improvements
   - Known limitations

## Output

Provide a comprehensive summary for review.
