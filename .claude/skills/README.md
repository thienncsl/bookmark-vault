# Skills

Specialized workflows and patterns for Bookmark Vault development.

## Available Skills

### Bookmark-Specific

| Skill | Description |
|-------|-------------|
| `bookmark-analyzer` | Multi-perspective codebase analysis with context isolation |
| `bookmark-checker` | Automated code quality checks (types, components, tests) |
| `bookmark-patterns` | Reference for Bookmark Vault coding patterns |
| `bookmark-safety` | Protects bookmark data from accidental deletion |
| `bookmark-validator` | Validates bookmark data quality (links, tags, duplicates) |

### Development Process

| Skill | Description |
|-------|-------------|
| `creating-docs` | Create documentation or guidance files |
| `managing-localstorage` | Handle localStorage CRUD operations in React apps |
| `reviewing-code` | Review code changes before merging |

## Usage

Skills are invoked automatically when relevant to your task. You can also invoke them directly using the Skill tool.
